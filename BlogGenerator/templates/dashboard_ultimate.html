📋 Step-by-Step Setup Guide
Step 1: Get API Key
  1. Log into Squarespace
  2. Go to Settings → Advanced → API Keys
  3. Click "Generate API Key"
  4. Name: "KnotStranded Generator"
  5. Copy key (starts with sqsp_)
  6. Paste in field below
Step 2: Get Site ID
  1. Settings → Advanced → Developer
  2. Copy your Site ID
  3. Paste in field below
Step 3: Get Collection ID
  1. Open your Blog page in editor
  2. Click Settings (gear icon)
  3. Look at URL: /config/pages/[ID]
  4. Copy the ID after /pages/
  5. Paste in field below
